# DigiDiary 📙✏️
A simple digital diary app made using Android Jetpack Compose

## ⚙️ Features
1. Add, View, Update and Delete Diary Entries
2. Persistant Storage with Room Database
3. Multi-Screen Navigation
4. Date and Time Picker
